```python
class Student(object):  #继承自 object类

    def __init__(self, name, score):
        self.name = name
        self.score = score

    def print_score(self):
        print('%s: %s' % (self.name, self.score))
``` 
* 访问限制
__ #普通变量名前加两个下划线就变成了私有变量
* 获取对象信息
 * type()
 * isinstance()
 * dir()
