#### 常用操作
/ ? 搜索 n 下一个 N 上一个
a 在光标后插入
x 删除光标所在位置字符
dd 删除整行
. 重复上一命令
N + command 重复N次该命令
N + G(shift+g) 到第N行
gg 到第一行
G 到最后一行
0 到行头
$ 到行尾
yy 复制一行
d 剪切
p 光标后粘贴
u 撤消

#### 基本操作
:set nu 显示行号
ctrl + o 光标回到之前位置

“ctrl+n” 代码提示
