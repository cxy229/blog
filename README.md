### post格式
```
---
layout     : post
title      : glossaries
categories : [English]
tags       : [notes]
---
```
- `timing: false` 默认文章在时间轴页面显示, false为不显示
- `life: true` 若life为true，文章改在生活页面显示
- `permalink: /life.html` 修改文章URL为固定URL
